# Software Architecture Blueprint 🏗️

This repository is a comprehensive collection of architectural patterns and best practices for building scalable, high-performance distributed systems with the **.NET ecosystem**.

### 🛠️ What's Inside?
- **Domain-Driven Design (DDD):** Strategic and tactical patterns.
- **Microservices Patterns:** Implementation of Saga, Outbox, and CQRS.
- **Messaging:** Integration with RabbitMQ and AWS SQS for asynchronous processing.
- **Persistence:** High-performance data access with SQL Server, Oracle, and MongoDB.
- **Cloud Native:** Docker-ready configurations and AWS integration (S3/SQS).

### 🚀 Purpose
This is my personal "reference manual" for designing systems that are not just functional, but resilient and maintainable. It serves as the solid foundation for my current research in **AI Engineering and RAG systems**.
